# my-life-style
```
目前預計這個工作結束後就不再從事軟體工程師行業，徹徹底底地退出！（是的
為什麼呢？說不上來的煩吧！

自己從高中開始自學程式以來，原本只是打算之後做遊戲，後來畢業發現，幹～台灣根本沒這產業鏈，
因為從小都是看日本動漫長大、跟韓國遊戲，還覺得之後工作可以至少碰到這產業，夢想直接破滅。
（應該休學一年先出來走跳的）
工作後發現NO，台灣只有科技業及資訊政府案，再來就是產業彼此內部競爭。

此外，在南部高雄薪資也不是特別高，同時要承擔不少壓力，28K或45K，差了17K，但是壓力小很多
，每天只要固定做重複的事，不用動腦，加上我是個躺平就不想坐起來的人，待在舒適圈才是生活的
王道。

此外，補充一下，30年前麥當勞大薯32元現在66元
那時的25k，試問現在應該是多少錢？
32:66 = 25000:x
x =  51000左右
薪水沒漲，要學的東西變多，消費變貴，房價不說了。

想了想，還是躺平好了！做簡單的工作有收入可以過日子就好，我的人生並不想被台灣社會綁架。
直接降低自己需求後：
每天只要工作4小時，睡眠10小時，10小時玩樂，一週工作上班4天，每年最多只工作180天。

看起來很廢，但想到光有人一生下來就是田橋子、大老闆董事小孩、收租仔，真的覺得我這樣的想法剛
好而已。不想一輩子被奴役，上面繼續吃香喝拉，動動手指做決策影響千萬人，可悲資本主義。

最後，我不想證明什麼，沒必要證明給NPC看，也沒有必要為NPC提供服務，我就是這世界上的觀察者
而已。

此外，真得結束後會嘗試去公園睡覺看看，畢竟只要睡成功一次，之後出遊住宿什麼也都不用看價格了。

這邊文只是單純紀錄我自己的想法
```
